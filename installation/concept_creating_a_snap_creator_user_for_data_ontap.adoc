= Data ONTAP용 Snap Creator 사용자 생성
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Snap Creator를 설치하기 전에 Data ONTAP용 Snap Creator 사용자를 생성해야 합니다. Snap Creator 사용자를 생성하기 위해 따르는 프로세스는 시스템이 7-Mode에서 Data ONTAP를 실행하는지 clustered Data ONTAP에서 실행하는지에 따라 달라집니다.
