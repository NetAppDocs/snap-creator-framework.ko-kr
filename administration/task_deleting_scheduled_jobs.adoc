= 예약된 작업을 삭제하는 중입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
예약된 작업을 삭제할 수 있습니다.

. Snap Creator GUI 기본 메뉴에서 * Management * > * Schedules * 를 선택합니다.
. 일정 탭의 예약된 작업 목록에서 작업을 선택하고 * 삭제 * 를 클릭합니다.
. 확인 창에서 * 예 * 를 클릭합니다.
+
예약된 작업이 목록에서 삭제됩니다.
