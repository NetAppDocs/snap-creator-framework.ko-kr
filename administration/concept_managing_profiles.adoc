= 프로파일 관리
:icons: font
:imagesdir: ../media/


[role="lead"]
프로파일을 작성, 보기 및 삭제할 수 있습니다.

프로필은 기본적으로 구성 파일을 구성하는 데 사용되는 폴더입니다. 프로파일은 역할 기반 액세스 제어(RBAC)에 대한 객체 역할도 합니다. 즉, 사용자가 특정 프로파일 및 내에 포함된 구성 파일에 대한 액세스를 허용할 수 있습니다.
