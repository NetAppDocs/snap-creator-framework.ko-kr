= 예약된 작업 실행 중
:icons: font
:imagesdir: ../media/


[role="lead"]
예약된 작업을 실행할 수 있습니다.

. Snap Creator GUI 기본 메뉴에서 * Management * > * Schedules * 를 선택합니다.
. 일정 탭의 예약된 작업 목록에서 작업을 선택하고 * 실행 * 을 클릭합니다.
